# 項目表

## 目錄用途

- `worker-send/`：Cloudflare Email Worker 專案
- `docker-accept/`：接收 webhook 的 Node.js / TypeScript 服務
- `single-file/`：issue #12 的 Python / Node.js / Java 單文件部署版本
- `.github/workflows/`：CI/CD，自動建置 Docker 映像
- `agent/`：本次接手紀錄與任務資訊

## 依賴概覽

### worker-send

- `wrangler`：Cloudflare Worker 開發與部署
- `typescript`：型別與編譯

### docker-accept

- `express`：HTTP API 與靜態頁面
- `typescript`：型別與編譯
- `tsx`：本地開發啟動

### single-file

- `python/webhook_mail.py`：Python 標準庫 HTTP server，JSON 檔案持久化
- `node/webhook-mail.js`：Node.js 內建 `http` 服務，JSON 檔案持久化
- `java/WebhookMail.java`：JDK 內建 `HttpServer`，JSON 檔案持久化，內含輕量 JSON parser / writer
